# MCP SDK API Findings — php-mcp/server v3.3.0

## Installation

```
composer require php-mcp/server
```

Paket: `php-mcp/server ^3.3` (mit `-W` für Dependency-Updates)

---

## Server aufbauen: Builder-Pattern

Kein `new Server(...)` direkt — stattdessen `Server::make()` → `ServerBuilder`.

```php
$server = Server::make()
    ->withServerInfo('My Server', '1.0.0')
    ->withTool(callable $handler, ?string $name, ?string $description)
    ->build();
```

---

## `withServerInfo()`

```php
ServerBuilder::withServerInfo(string $name, string $version): self
```

- Kein `setServerInfo()` — heißt `withServerInfo()`
- Pflichtfeld — ohne es wirft `build()` eine `ConfigurationException`

---

## `withTool()`

```php
ServerBuilder::withTool(
    callable|array|string $handler,
    ?string $name = null,
    ?string $description = null,
    ?ToolAnnotations $annotations = null,
    ?array $inputSchema = null
): self
```

- Handler kann Closure, `[ClassName::class, 'methodName']` oder Callable-String sein
- `name` und `description` optional — werden aus Reflection/DocBlock ermittelt falls null
- `inputSchema` optional — wird automatisch aus PHP-Typen generiert falls null
- Rückgabewert des Handlers wird automatisch konvertiert:
  - `string`/`int`/`float`/`bool` → `TextContent`
  - `array` (kein Content) → JSON-kodiertes `TextContent`
  - `Content`-Objekte → direkt verwendet
  - `null` → `TextContent('(null)')`

## `StdioServerTransport`

```php
use PhpMcp\Server\Transports\StdioServerTransport;

$transport = new StdioServerTransport(); // STDIN/STDOUT defaults
$server->listen($transport); // blockierend — ReactPHP Event Loop
```

- Klasse: `PhpMcp\Server\Transports\StdioServerTransport`
- Constructor: `__construct($inputStreamResource = STDIN, $outputStreamResource = STDOUT)`
- Sendet/empfängt JSON-RPC über STDIN/STDOUT (newline-delimited)
- `listen()` ist **blockierend** (ReactPHP Loop)

---

## Tool-Handler Return-Format

Tool-Handler können zurückgeben:
- `string` → automatisch `TextContent`
- `array` → JSON-encoded `TextContent` (wenn kein Content-Array)
- `PhpMcp\Schema\Content\Content` (z.B. `TextContent::make(...)`) → direkt
- Beliebiges Objekt → JSON-encoded `TextContent`

Empfehlung für unsere Tools: einfach `string` oder `array` zurückgeben — das SDK konvertiert automatisch.

---

## Vollständiges Beispiel

```php
use PhpMcp\Server\Server;
use PhpMcp\Server\Transports\StdioServerTransport;

$server = Server::make()
    ->withServerInfo('PhpCodeArcheology', '2.0.0')
    ->withTool(
        handler: function (string $path): string {
            return "Analysis result for $path";
        },
        name: 'analyze',
        description: 'Analyze PHP code'
    )
    ->build();

$server->listen(new StdioServerTransport());
```
