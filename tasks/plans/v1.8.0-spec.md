# v1.8.0 — Knowledge Graph Visualisierung: Implementierungs-Spezifikation

## (a) Wie Graph-Daten in das Template kommen

### Datenfluss: GraphDataProvider → HtmlReport → Twig

**Pattern (analog zu `generateGitPage()`):**

```php
// In HtmlReport.php → generateReportFiles():
[$this, 'generateKnowledgeGraphPage'],

// Neue Methode:
protected function generateKnowledgeGraphPage(): void
{
    $data = $this->dataProviderFactory->getGraphDataProvider()->getTemplateData();
    $data['pageTitle'] = 'Knowledge Graph';
    $data['currentPage'] = 'knowledge-graph.html';
    $this->renderTemplate('knowledge-graph.html.twig', $data, 'knowledge-graph.html');
}
```

**Was `getTemplateData()` liefert:**

`GraphDataProvider::gatherData()` befüllt `$this->templateData['graphData']` via `getGraphData()`:

**Inline JSON im Template (Pflicht-Pattern):**

```twig
<script>const graphData = {{ graphJson|raw }};</script>
```

Das `graphJson` wird in PHP vor dem Template-Render erzeugt:

```php
protected function generateKnowledgeGraphPage(): void
{
    $data = $this->dataProviderFactory->getGraphDataProvider()->getTemplateData();
    $data['graphJson'] = json_encode($data['graphData'] ?? [], JSON_UNESCAPED_UNICODE | JSON_UNESCAPED_SLASHES);
    unset($data['graphData']); // nicht als PHP-Array ans Template
    $data['pageTitle'] = 'Knowledge Graph';
    $data['currentPage'] = 'knowledge-graph.html';
    $this->renderTemplate('knowledge-graph.html.twig', $data, 'knowledge-graph.html');
}
```

## (b) D3.js Integration-Plan

### Download

```bash
curl -o templates/html/assets/js/d3.min.js https://cdn.jsdelivr.net/npm/d3@7/dist/d3.min.js
```

### Asset-Einbindung

D3 wird **nur** in `knowledge-graph.html.twig` eingebunden — **nicht** global in `header.html.twig`.

```twig
{# Am Ende des Templates, vor dem footer-Include: #}
<script src="{% if isSubdir %}../{% endif %}assets/js/d3.min.js"></script>
<script>
  // Graph-Logik direkt inline (oder separates knowledge-graph.js)
  const graphData = {{ graphJson|raw }};
  // D3-Code hier...
</script>
```

**Alternative mit separatem JS-File** (empfohlen für Übersichtlichkeit):

```twig
<script src="{% if isSubdir %}../{% endif %}assets/js/d3.min.js"></script>
<script src="{% if isSubdir %}../{% endif %}assets/js/knowledge-graph.js"></script>
<script>initKnowledgeGraph({{ graphJson|raw }});</script>
```

→ Datei: `templates/html/assets/js/knowledge-graph.js` — exportiert `function initKnowledgeGraph(data)`

### FileCopier

`d3.min.js` und `knowledge-graph.js` liegen in `templates/html/assets/js/` und werden automatisch
durch den bestehenden `FileCopier`-Block in `HtmlReport::generate()` mitkopiert:

```php
$fc->setFiles([$this->templateDir . 'assets']);
$fc->copyFilesTo(rtrim($this->outputDir . '/assets', DIRECTORY_SEPARATOR));
```

### D3 Force-Directed Graph — `knowledge-graph.js` Struktur

```javascript
(function() {
  // --- Farbschemata ---
  const NODE_COLORS = {
    class:     '#06b6d4',
    interface: '#a855f7',
    trait:     '#f97316',
    package:   '#22c55e',
    author:    '#ec4899',
    method:    '#94a3b8',
    function:  '#64748b',
  };

  const EDGE_COLORS = {
    depends_on:  { dark: 'rgba(255,255,255,0.2)',  light: 'rgba(0,0,0,0.15)',    dash: null,    width: 1 },
    extends:     { dark: '#3b82f6',                light: '#3b82f6',             dash: null,    width: 2 },
    implements:  { dark: '#a855f7',                light: '#a855f7',             dash: null,    width: 1.5 },
    uses_trait:  { dark: '#f97316',                light: '#f97316',             dash: null,    width: 1.5 },
    cycle_member:{ dark: '#ef4444',                light: '#ef4444',             dash: [5, 4],  width: 2 },
    belongs_to:  { dark: 'rgba(255,255,255,0.1)',  light: 'rgba(0,0,0,0.08)',    dash: [2, 3],  width: 1 },
    authored_by: { dark: 'rgba(236,72,153,0.3)',   light: 'rgba(236,72,153,0.4)',dash: null,    width: 1 },
    declares:    { dark: 'rgba(255,255,255,0.08)', light: 'rgba(0,0,0,0.06)',    dash: null,    width: 0.5 },
  };

  // --- Node-Radius (skaliert nach CC) ---
  function nodeRadius(node) {
    const cc = node.metrics?.cc ?? 1;
    return Math.max(5, Math.min(25, 5 + Math.sqrt(cc) * 2.5));
  }

  // --- Typ-Bestimmung (flags auswerten) ---
  function resolveNodeType(node) {
    if (node.type !== 'class') return node.type;
    if (node.flags?.interface) return 'interface';
    if (node.flags?.trait) return 'trait';
    return 'class';
  }

  // --- URL für Klick auf Klassen-Node ---
  function nodeUrl(node) {
    if (node.type === 'class') {
      return 'classes/' + node.id.replace('class:', '') + '.html';
    }
    return null;
  }

  // --- Hauptfunktion ---
  function initKnowledgeGraph(data) {
    const isLight = document.documentElement.getAttribute('data-theme') === 'light';

    const svg = d3.select('#knowledge-graph-svg');
    const container = document.getElementById('graph-container');
    const width = container.clientWidth;
    const height = container.clientHeight;

    // Stat-Counter befüllen
    document.getElementById('stat-nodes').textContent = data.nodes.length;
    document.getElementById('stat-edges').textContent = data.edges.length;
    document.getElementById('stat-cycles').textContent = data.cycles.length;
    document.getElementById('stat-packages').textContent =
      data.nodes.filter(n => n.type === 'package').length;

    // Filter-State
    const activeNodeTypes = new Set(Object.keys(NODE_COLORS));
    const activeEdgeTypes = new Set(Object.keys(EDGE_COLORS));
    let minCC = 0;
    let problemsOnly = false;

    // Cycle-Node-Set vorberechnen
    const cycleNodeSet = new Set(data.cycles.flatMap(c => c.nodes));

    // D3 Setup
    const g = svg.append('g');

    const zoom = d3.zoom()
      .scaleExtent([0.05, 4])
      .on('zoom', (event) => g.attr('transform', event.transform));
    svg.call(zoom);

    // Arrow-Marker für gerichtete Edges
    const defs = svg.append('defs');
    Object.entries(EDGE_COLORS).forEach(([type, style]) => {
      defs.append('marker')
        .attr('id', 'arrow-' + type)
        .attr('viewBox', '0 -4 8 8')
        .attr('refX', 14)
        .attr('refY', 0)
        .attr('markerWidth', 5)
        .attr('markerHeight', 5)
        .attr('orient', 'auto')
        .append('path')
        .attr('d', 'M0,-4L8,0L0,4')
        .attr('fill', isLight ? style.light : style.dark);
    });

    // Simulation
    let simulation;
    let linkSelection, nodeSelection;

    function buildGraph() {
      // Filter anwenden
      const visibleNodes = data.nodes.filter(n => {
        const t = resolveNodeType(n);
        if (!activeNodeTypes.has(t)) return false;
        const cc = n.metrics?.cc ?? 0;
        if (cc < minCC) return false;
        if (problemsOnly && (n.problems?.length ?? 0) === 0) return false;
        return true;
      });

      const visibleNodeIds = new Set(visibleNodes.map(n => n.id));

      const visibleEdges = data.edges.filter(e =>
        activeEdgeTypes.has(e.type) &&
        visibleNodeIds.has(e.source.id ?? e.source) &&
        visibleNodeIds.has(e.target.id ?? e.target)
      );

      // Alte SVG-Inhalte (außer defs) löschen
      g.selectAll('*').remove();

      // Edges
      linkSelection = g.append('g')
        .attr('class', 'edges')
        .selectAll('line')
        .data(visibleEdges)
        .join('line')
        .attr('stroke', e => {
          const style = EDGE_COLORS[e.type] ?? EDGE_COLORS.depends_on;
          return isLight ? style.light : style.dark;
        })
        .attr('stroke-width', e => EDGE_COLORS[e.type]?.width ?? 1)
        .attr('stroke-dasharray', e => {
          const dash = EDGE_COLORS[e.type]?.dash;
          return dash ? dash.join(',') : null;
        })
        .attr('marker-end', e => 'url(#arrow-' + e.type + ')')
        .attr('opacity', 0.7);

      // Nodes
      const nodeGroup = g.append('g')
        .attr('class', 'nodes')
        .selectAll('g')
        .data(visibleNodes)
        .join('g')
        .attr('class', 'node')
        .style('cursor', n => nodeUrl(n) ? 'pointer' : 'default')
        .call(d3.drag()
          .on('start', dragStarted)
          .on('drag', dragged)
          .on('end', dragEnded));

      nodeGroup.append('circle')
        .attr('r', n => nodeRadius(n))
        .attr('fill', n => NODE_COLORS[resolveNodeType(n)] ?? '#6b7280')
        .attr('fill-opacity', 0.85)
        .attr('stroke', n => cycleNodeSet.has(n.id) ? '#ef4444' : 'rgba(255,255,255,0.15)')
        .attr('stroke-width', n => cycleNodeSet.has(n.id) ? 2.5 : 1);

      nodeGroup.append('text')
        .attr('dy', n => nodeRadius(n) + 10)
        .attr('text-anchor', 'middle')
        .attr('font-size', '9px')
        .attr('fill', isLight ? '#334155' : 'rgba(255,255,255,0.6)')
        .text(n => {
          const name = n.name ?? '';
          const parts = name.split('\\');
          return parts[parts.length - 1]; // nur einfacher Klassenname
        });

      nodeSelection = nodeGroup;

      // Tooltip
      const tooltip = document.getElementById('graph-tooltip');
      nodeGroup
        .on('mouseover', (event, n) => {
          const t = resolveNodeType(n);
          tooltip.style.display = 'block';
          tooltip.innerHTML = `
            <div class="graph-tooltip-title" style="color:${NODE_COLORS[t] ?? '#fff'}">${n.name}</div>
            <div class="graph-tooltip-row"><span>Type</span><span>${t}</span></div>
            ${n.metrics?.cc !== undefined ? `<div class="graph-tooltip-row"><span>CC</span><span>${n.metrics.cc}</span></div>` : ''}
            ${n.metrics?.lcom !== undefined ? `<div class="graph-tooltip-row"><span>LCOM</span><span>${n.metrics.lcom}</span></div>` : ''}
            ${n.metrics?.mi !== undefined ? `<div class="graph-tooltip-row"><span>MI</span><span>${n.metrics.mi?.toFixed(1)}</span></div>` : ''}
            ${n.problems?.length > 0 ? `<div class="graph-tooltip-row"><span>Problems</span><span style="color:#ef4444">${n.problems.length}</span></div>` : ''}
            ${cycleNodeSet.has(n.id) ? `<div style="color:#ef4444; margin-top:0.25rem; font-size:0.75rem;">⚠ In Dependency Cycle</div>` : ''}
          `;
        })
        .on('mousemove', (event) => {
          const rect = container.getBoundingClientRect();
          let x = event.clientX - rect.left + 12;
          let y = event.clientY - rect.top + 12;
          if (x + 220 > rect.width) x -= 240;
          tooltip.style.left = x + 'px';
          tooltip.style.top = y + 'px';
        })
        .on('mouseout', () => { tooltip.style.display = 'none'; })
        .on('click', (event, n) => {
          const url = nodeUrl(n);
          if (url) window.location.href = url;
        });

      // Simulation neu starten
      if (simulation) simulation.stop();
      simulation = d3.forceSimulation(visibleNodes)
        .force('link', d3.forceLink(visibleEdges)
          .id(d => d.id)
          .distance(80)
          .strength(0.3))
        .force('charge', d3.forceManyBody().strength(-120))
        .force('center', d3.forceCenter(width / 2, height / 2))
        .force('collision', d3.forceCollide().radius(n => nodeRadius(n) + 4))
        .on('tick', ticked);
    }

    function ticked() {
      if (linkSelection) {
        linkSelection
          .attr('x1', d => d.source.x)
          .attr('y1', d => d.source.y)
          .attr('x2', d => d.target.x)
          .attr('y2', d => d.target.y);
      }
      if (nodeSelection) {
        nodeSelection.attr('transform', d => `translate(${d.x},${d.y})`);
      }
    }

    function dragStarted(event, d) {
      if (!event.active) simulation.alphaTarget(0.3).restart();
      d.fx = d.x; d.fy = d.y;
    }
    function dragged(event, d) {
      d.fx = event.x; d.fy = event.y;
    }
    function dragEnded(event, d) {
      if (!event.active) simulation.alphaTarget(0);
      d.fx = null; d.fy = null;
    }

    // --- Filter-Event-Listener ---
    document.querySelectorAll('.filter-chip[data-filter="node-type"]').forEach(chip => {
      chip.addEventListener('click', () => {
        const val = chip.dataset.value;
        if (activeNodeTypes.has(val)) {
          activeNodeTypes.delete(val);
          chip.classList.add('inactive');
        } else {
          activeNodeTypes.add(val);
          chip.classList.remove('inactive');
        }
        buildGraph();
      });
    });

    document.querySelectorAll('.filter-chip[data-filter="edge-type"]').forEach(chip => {
      chip.addEventListener('click', () => {
        const val = chip.dataset.value;
        if (activeEdgeTypes.has(val)) {
          activeEdgeTypes.delete(val);
          chip.classList.add('inactive');
        } else {
          activeEdgeTypes.add(val);
          chip.classList.remove('inactive');
        }
        buildGraph();
      });
    });

    const ccSlider = document.getElementById('cc-filter');
    const ccLabel = document.getElementById('cc-filter-value');
    ccSlider?.addEventListener('input', () => {
      minCC = parseInt(ccSlider.value);
      ccLabel.textContent = '≥ ' + minCC;
      buildGraph();
    });

    document.getElementById('problems-only')?.addEventListener('change', (e) => {
      problemsOnly = e.target.checked;
      buildGraph();
    });

    document.getElementById('graph-reset-zoom')?.addEventListener('click', () => {
      svg.transition().duration(300).call(zoom.transform, d3.zoomIdentity);
    });

    // Initial render
    buildGraph();
  }

  // Global verfügbar machen (wird vom inline-Script aufgerufen)
  window.initKnowledgeGraph = initKnowledgeGraph;
})();
```

## (e) Farbschema für Node- und Edge-Typen

### Node-Typ-Farben

| Typ        | Farbe (Dark)          | Farbe (Light)         | Tailwind-Referenz   |
|------------|-----------------------|-----------------------|---------------------|
| `class`    | `#06b6d4`             | `#0891b2`             | cyan-500 / cyan-600 |
| `interface`| `#a855f7`             | `#9333ea`             | purple-500          |
| `trait`    | `#f97316`             | `#ea580c`             | orange-500          |
| `package`  | `#22c55e`             | `#16a34a`             | green-500           |
| `author`   | `#ec4899`             | `#db2777`             | pink-500            |
| `method`   | `#94a3b8`             | `#64748b`             | slate-400 / slate-500|
| `function` | `#64748b`             | `#475569`             | slate-500           |

**Besonderheiten:**
- Nodes in Dependency Cycles erhalten einen roten Stroke (`#ef4444`, 2.5px)
- Abstract-Classes: leicht desaturiert (opacity 0.7 auf dem Fill)
- Node-Größe = `max(5, min(25, 5 + sqrt(cc) * 2.5))` Pixel Radius

### Edge-Typ-Farben

| Typ          | Dark-Mode                     | Light-Mode                   | Dash     | Width |
|--------------|-------------------------------|------------------------------|----------|-------|
| `depends_on` | `rgba(255,255,255,0.2)`       | `rgba(0,0,0,0.15)`           | —        | 1px   |
| `extends`    | `#3b82f6` (blue-500)          | `#3b82f6`                    | —        | 2px   |
| `implements` | `#a855f7` (purple-500)        | `#a855f7`                    | —        | 1.5px |
| `uses_trait` | `#f97316` (orange-500)        | `#f97316`                    | —        | 1.5px |
| `cycle_member`| `#ef4444` (red-500)          | `#ef4444`                    | `5,4`    | 2px   |
| `belongs_to` | `rgba(255,255,255,0.1)`       | `rgba(0,0,0,0.08)`           | `2,3`    | 1px   |
| `authored_by`| `rgba(236,72,153,0.3)`        | `rgba(236,72,153,0.4)`       | —        | 1px   |
| `declares`   | `rgba(255,255,255,0.08)`      | `rgba(0,0,0,0.06)`           | —        | 0.5px |

### Design-Prinzipien

1. **Cyan als Primärfarbe** — bleibt dem bestehenden Design-System treu (`#06b6d4`)
2. **Semantische Farben** — Rot für Fehler/Cycles, Grün für Packages, Lila für Interfaces (konsistent mit bestehenden Badge-Farben im Report)
3. **Transparenz-basierte Subtilität** für unwichtige Kanten (`depends_on`, `belongs_to`, `declares`) — Graph bleibt lesbar
4. **Light-Theme-Kompatibilität** — alle Edge-Farben haben Light-Mode-Variante; Node-Farben bleiben gleich (hoch gesättigte Farben funktionieren auf hellem und dunklem Hintergrund)
