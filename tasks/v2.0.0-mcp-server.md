## Ausgangslage

### MCP-Protokoll (Model Context Protocol)

MCP ist ein offenes Protokoll das AI-Tools nutzen um externe Datenquellen und Tools anzubinden:

- **Transport:** JSON-RPC 2.0 über **stdio** (stdin/stdout)
- **Lifecycle:** Client sendet `initialize` → Server antwortet mit Capabilities → Client ruft `tools/list` → Server listet verfügbare Tools → Client ruft `tools/call` mit Tool-Name und Parametern
- **Responses:** JSON-RPC Responses mit `result` oder `error`
- **Kein HTTP nötig** — reines stdin/stdout Protokoll

### JSON-RPC 2.0 Format

```json
// Request
{"jsonrpc": "2.0", "id": 1, "method": "tools/call", "params": {"name": "get_health_score", "arguments": {}}}

// Success Response
{"jsonrpc": "2.0", "id": 1, "result": {"content": [{"type": "text", "text": "..."}]}}

// Error Response
{"jsonrpc": "2.0", "id": 1, "error": {"code": -32602, "message": "Invalid params"}}
```

### Bestehende Architektur

```
bin/phpcodearcheology          → Entry Point (CLI)
src/Application/Application.php → run(), runAnalysis(), dispatchCommand()
src/Application/ArgumentParser.php → CLI-Arg Parsing, dispatchCommand() für Subcommands
src/Application/Config.php     → Konfiguration
src/Application/Analyzer.php   → Analyse-Executor

MetricsController              → Zentraler Daten-Hub nach Analyse
DataProviderFactory            → Erstellt alle DataProviders
  ├── ProjectDataProvider      → Health Score, Projekt-Metriken
  ├── ProblemDataProvider      → Probleme mit Level und Message
  ├── ClassDataProvider        → Klassen-Metriken
  ├── GitDataProvider          → Git-Hotspots, Churn × Complexity
  ├── RefactoringPriorityDataProvider → Priorisierte Refactoring-Kandidaten
  ├── ClassCouplingDataProvider → Dependencies
  └── GraphDataProvider        → Knowledge Graph (seit v1.7.0)
```

### Bestehende Subcommands

`Application.php` hat bereits `dispatchCommand()` das Subcommands verteilt:
- `init` — Erstellt Config-Datei
- `compare` — Vergleicht zwei Analysen
- `baseline` — Erstellt Baseline

Neuer Subcommand `mcp` wird hier eingehängt.
